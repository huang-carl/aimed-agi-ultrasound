# 阿尔麦德 AI 诊断系统技术白皮书

## 一、系统概述

阿尔麦德 AI 诊断系统是基于口服超声造影剂+AI 多智能体诊断的胃胰疾病早筛平台，是全球首创的口服超声造影 AI 辅助诊断系统。

## 二、核心技术

### 2.1 图像分割技术

- **模型：** SAM（Segment Anything Model）ViT-B
- **参数量：** 90M
- **分割精度：** Dice 系数 0.85+
- **推理速度：** <2s/张

### 2.2 诊断分类技术

- **模型：** 多模态融合模型
- **输入：** 超声图像 + 临床文本
- **输出：** 诊断建议 + 置信度
- **准确率：** 80%+

### 2.3 向量检索技术

- **引擎：** ChromaDB
- **文档数量：** 1000+ 医疗文档
- **检索速度：** <100ms
- **准确率：** 90%+

## 三、系统架构

### 3.1 前端架构

- **框架：** 原生 HTML/CSS/JS
- **部署：** Nginx + HTTPS
- **响应式：** 支持 PC/平板/手机

### 3.2 后端架构

- **框架：** FastAPI + Uvicorn
- **端口：** 18790
- **AI 模型：** 阿里云百炼 + NVIDIA NIM
