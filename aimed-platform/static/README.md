## 知识库分类

各公司知识库包含以下目录：
- `company/` — 公司基本信息、营业执照
- `product/` — 产品手册、技术规格
- `patents/` — 专利证书
- `certificates/` — 资质证书
- `sales_materials/` — 销售话术、客户案例
- `technical_docs/` — 技术文档
- `ai_system/` — AI系统文档（阿尔麦德专属）
