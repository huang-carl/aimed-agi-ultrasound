# AIMED 智慧医疗服务生态 — 部署说明

## 系统简介

AIMED 智慧医疗服务生态是一个面向医疗健康产业的**多公司统一管理平台**，支持 5 家成员企业独立运营、数据隔离，由"数字总经理"统一管控。

## 文件结构

```
aimed_ecosystem/
├── platform/
│   └── index.html          ← 管理平台主文件（直接双击打开）
├── company/                ← 各公司知识库
│   ├── 东亚医药/
│   ├── 国械恒发/
│   ├── 阿尔麦德/
│   ├── 阿尔麦德上海/
│   └── 和七汇企业/
├── ecosystem/              ← 生态总览文档
├── database/               ← 文档索引
├── uploads/                ← 上传目录
└── README.md               ← 使用指南
```

## 快速启动

### 方式一：本地直接运行（推荐）

1. 进入 `platform` 文件夹
2. 双击 `index.html`
3. 使用浏览器打开即可

> 无需安装任何服务器或依赖，纯前端单文件应用。

### 方式二：局域网访问

将 `aimed_ecosystem` 文件夹放到局域网共享路径，其他设备通过 `\\电脑名\aimed_ecosystem\platform\index.html` 访问。

### 方式三：部署到 Web 服务器

将整个 `aimed_ecosystem` 文件夹上传至 nginx/apache/iis 等 Web 服务器目录，通过域名访问 `platform/index.html`。

## 功能模块

- **工作台**：各公司快速入口
- **知识库管理**：上传/删除/分类文档（营业执照、专利、证书、产品手册、销售资料等）
- **用户管理**：增删改用户、分配角色和公司
- **超级管理**（仅 gm）：全局用户统计、公司管理、生态设置

## 数据存储

- 所有数据存储在浏览器 **LocalStorage** 中
- 建议定期导出 `platform/index.html` 作为备份
- 切换浏览器或清除缓存后数据将丢失

## 浏览器兼容性

- Chrome 90+
- Edge 90+
- Firefox 88+
- Safari 14+

> 不支持 IE。

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务
- 无需数据库
- 无需安装依赖
