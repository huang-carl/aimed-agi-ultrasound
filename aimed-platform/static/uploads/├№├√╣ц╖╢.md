# 文件命名规范（AI自动重命名）

## 命名格式

`[公司]-[类型]-[名称]-[版本]-[日期].扩展名`

### 公司代码
| 代码 | 公司 |
|------|------|
| `DY` | 东亚医药 |
| `GHF` | 国械恒发 |
| `AME` | 阿尔麦德 |
| `ECO` | 生态系统通用 |

### 文件类型
| 代码 | 类型 |
|------|------|
| `BIZ` | 营业执照/注册文件 |
| `CERT` | 证书/资质 |
| `PAT` | 专利文件 |
| `PRO` | 产品资料 |
| `SAL` | 销售资料 |
| `TEC` | 技术文档 |
| `AI` | AI/软件系统 |
| `CON` | 合同协议 |
| `REP` | 报告/分析 |
| `MKT` | 市场/推广 |
| `OTH` | 其他 |

### 命名示例
```
DY-BIZ-天下牌营业执照-2026-V1-20260429.pdf
GHF-PAT-胃肠造影剂专利-CNXXXX-V2-20260429.pdf
AME-AI-AIMED系统白皮书-V1-20260429.pdf
ECO-CERT-医疗器械注册证-GHF001-V1-20260429.pdf
```

### 归档规则
- 上传后进入 `uploads/待处理/` 目录
- AI分析内容后自动重命名
- 用户确认后移动至 `uploads/已归档/[公司]/[类型]/` 目录
- 同时写入 `database/documents.json` 数据库
