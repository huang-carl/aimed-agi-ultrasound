## 项目概述

**充盈视界（FillingVision）** 是基于 AMD Ryzen AI 的上消化道无创诊断平台，融合口服超声造影剂 + 充盈超声检查法+AI 影像诊断算法，为基层医疗机构提供消化系统疾病无创早筛方案。

## 核心功能

### 🎯 双器官联合诊断
- **胰腺诊断** - 国内唯一口服胰腺超声造影剂注册证支持
- **胃部诊断** - 早期筛查无创低成本手段

### 🤖 AIMED Agent Swarm
- 18 个 AI Agent 协同工作
- 72B+7B+14B 分层模型配置
- 量化 + 蒸馏+MOE 三大瘦身技术
