# 项目概览

该项目旨在通过前沿的人工智能和超声技术，提供精准的医疗解决方案。我们致力于提高医疗工作效率和病患的治疗效果。

# 快速开始

1. 克隆库：
   ```bash
   git clone https://github.com/huang-carl/aimed-agi-ultrasound.git
   cd aimed-agi-ultrasound
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动项目：
   ```bash
   npm start
   ```
