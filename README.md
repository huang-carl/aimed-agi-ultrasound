项目概述 | Project Overview
 
本项目依托口服超声造影剂与充盈超声造影检查核心技术，结合深度学习与多智能体AI算法，打造专业化上消化道无创智能诊断平台，聚焦胃部、胰腺等关键器官病变早期筛查与辅助诊断，为基层医疗机构提供高效、低成本、易部署的消化道疾病早筛解决方案，推动医疗AI技术临床普惠化落地。
 
Relying on core technologies of oral ultrasound contrast agent and filling ultrasound contrast examination, combined with deep learning and multi-agent AI algorithms, this project develops a professional non-invasive intelligent diagnosis platform for the upper gastrointestinal tract. It focuses on early screening and auxiliary diagnosis of lesions in key organs such as the stomach and pancreas, providing primary medical institutions with an efficient, low-cost and easy-to-deploy solution for early gastrointestinal disease screening, and promoting the clinical inclusive implementation of medical AI technology.
