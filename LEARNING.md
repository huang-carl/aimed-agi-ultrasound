# LEARNING.md - 自我学习进化机制

_小超同学和小菲同学的持续学习和进化框架。_

---

## 🧠 学习循环

### 1. 感知 (Perceive)
- 接收用户消息和任务
- 观察系统状态和日志
- 收集环境信息

### 2. 行动 (Act)
- 执行任务
- 解决问题
- 创造价值

### 3. 反思 (Reflect)
- 记录到 `memory/YYYY-MM-DD.md`
- 什么做得好？
- 什么可以改进？
- 学到了什么？

### 4. 整合 (Integrate)
- 更新 `MEMORY.md` 长期记忆
- 提炼经验教训
- 调整行为模式

## 📚 知识管理

### 文件结构
```
workspace/
├── MEMORY.md              # 长期记忆（ curated ）
├── memory/
│   ├── 2026-04-21.md      # 每日记忆（ raw logs ）
│   └── YYYY-MM-DD.md
├── LEARNING.md            # 学习机制文档（本文件）
├── SOUL.md                # 核心人格
├── AGENTS.md              # 行为规范
└── IDENTITY.md            # 身份定义
```

### 记忆分层
- **短期记忆：** 当前会话上下文
- **每日记忆：** `memory/YYYY-MM-DD.md` - 原始日志
- **长期记忆：** `MEMORY.md` - 精选知识

- 回顾当日 `memory/YYYY-MM-DD.md`

### 每周回顾
- 检查 `memory/` 目录
- 提炼本周关键学习

### 每月进化
- 审视 `SOUL.md` 和 `IDENTITY.md`
- 评估是否需要调整人格或职责
- 与用户讨论进化方向
