# Skill: diagnostic-report-standard

## 描述
标准化超声造影诊断报告模板，支持胃/胰腺分型，确保报告格式统一、内容完整。

## 触发条件
- 用户请求生成诊断报告
- 调用 `/api/v1/diagnosis` 接口
- 需要输出结构化诊断结论时

## 报告模板

### 基础信息
```
【超声造影 AI 辅助分析报告】

检查日期：YYYY-MM-DD HH:MM
检查设备：[超声设备型号]
造影剂：[口服超声造影剂名称及剂量]
检查部位：[胃部/胰腺/其他]
患者 ID：[脱敏处理]
```

### 图像分析结果
```
## 一、灰阶超声表现
- 胃壁/胰腺形态：[描述]
- 病变位置：[精确解剖定位]
- 病变大小：[长×宽×高，mm]
- 回声特征：[低/等/高回声，均匀/不均匀]

## 二、超声造影表现
- 增强模式：[早增强/同步增强/延迟增强/无增强]
- 增强强度：[高/等/低增强]
- 增强均匀性：[均匀/不均匀/环状]
- 廓清模式：[早廓清/晚廓清/无廓清/向心填充]
- 伪足征：[有/无]
- 辐射状血管：[有/无]
```

### AI 辅助诊断结论
```
## 三、AI 诊断建议
**诊断分型：** [I-V 型，根据良恶性概率]
**恶性概率：** XX%（基于 XXXX 例训练集验证）
**关键特征：** [列出 2-3 个决定性特征]

**建议：**
- [ ] 定期随访（3-6 个月）
- [ ] 进一步检查（增强 CT/MRI）
- [ ] 穿刺活检
- [ ] 手术评估
```

### 免责声明
```
## 四、声明
本报告由 AI 辅助生成，仅供参考，不能替代专业医师诊断。
最终诊断请以临床医师综合判断为准。

报告生成时间：YYYY-MM-DD HH:MM:SS
AI 模型版本：Hermes V2.1
```

## 输出格式
- 默认输出 Markdown 格式
- 支持导出 PDF/Word（通过 `/api/v1/report/export`）
- 关键结论加粗，便于快速阅读

## 注意事项
1. **合规边界**：报告必须包含免责声明，严禁对外宣称"诊断"
2. **数据脱敏**：患者信息必须脱敏处理
3. **版本控制**：每次报告生成记录模型版本和日期
4. **人工复核**：AI 结论需医师签字确认后方可生效
