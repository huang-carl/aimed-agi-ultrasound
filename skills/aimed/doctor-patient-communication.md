# Skill: doctor-patient-communication

## 描述
医患沟通话术模板，支持专业模式（医生间交流）和通俗模式（医患沟通）双模式切换。

## 触发条件
- 用户请求生成医患沟通内容
- 需要向非专业人士解释诊断结果
- 医生间专业交流场景

## 模式切换

### 模式 1：专业模式（医生间交流）
**特点：** 医学术语、数据支撑、简洁高效

```
【专业模式示例】

患者，男，52 岁，上腹痛 3 月。
超声造影提示：胃窦部不规则增厚（约 3.2×2.1cm），
动脉期早增强，门脉期快速廓清，考虑胃恶性肿瘤可能（Type IV）。
建议：增强 CT 进一步评估，MDT 讨论。

关键指标：
- 恶性概率：78%
- 增强模式：早增强 + 早廓清
- 伪足征：阳性
```

### 模式 2：通俗模式（医患沟通）
**特点：** 通俗语言、安抚情绪、重点突出

```
【通俗模式示例】

您好，根据检查结果，我们发现您的胃部有一个约 3 厘米的增厚区域。
这个区域在注射造影剂后表现有些异常，需要进一步检查确认。

目前的情况：
✅ 好消息：发现得比较早，区域还比较局限
⚠️ 需要注意：异常表现比较明显，建议尽快做进一步检查

下一步建议：
1. 做一个增强 CT，更清楚地看看这个区域的情况
2. 带上所有检查结果，找消化科/普外科专家看看
3. 不要过于担心，很多胃部问题都是可以治疗的

有任何疑问随时问我。
```

## 话术模板库

### 解释诊断结果
```
专业版：
"超声造影显示 [部位] 存在 [特征]，考虑 [诊断分型]，
恶性概率 XX%，建议 [下一步行动]。"

通俗版：
"检查发现 [部位] 有个 [大小] 的 [描述]，
这个区域的表现 [正常/有些异常]，
我们建议 [下一步行动]，不要太担心。"
```

### 解释超声造影
```
专业版：
"口服超声造影剂后，实时观察胃壁/胰腺微循环灌注情况，
通过增强模式和廓清特征辅助鉴别良恶性病变。"

通俗版：
"您喝了一种特殊的'造影水'，它能让我们的超声波更清楚地看到
胃壁和胰腺的血流情况。就像给器官做了个'血液地图'，
帮助医生判断有没有异常区域。"
```

### 安抚情绪
```
"我理解您的担心，这是很正常的反应。
目前我们发现了问题，并且有明确的下一步计划，
这本身就是积极的一步。很多胃部问题都是可以治疗的，
我们一起把情况弄清楚。"
```

## 注意事项
1. **模式识别**：根据对话对象自动切换模式（医生/患者）
2. **情绪感知**：检测用户情绪状态，适当调整语气
3. **信息分级**：先给结论，再给细节，避免信息过载
4. **行动明确**：每次沟通必须包含明确的下一步建议
5. **合规提醒**：通俗模式下避免使用"癌症"等可能引起恐慌的词汇，用"异常"替代
