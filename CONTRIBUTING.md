# Contribution Guidelines for FillingVision

Thank you for your interest in contributing to the FillingVision project! We welcome contributions from the community. Please follow the guidelines below to ensure a smooth contribution process.

## Submitting Issues
- Before submitting an issue, please search existing issues to ensure that your issue hasn't already been reported.
- When submitting an issue, include the following information:
  - A descriptive title
  - Steps to reproduce the issue (if applicable)
  - Expected and actual behavior
  - Screenshots or logs (if relevant)

## Forking the Repository
1. Click the "Fork" button at the top right of the repository page to create a copy of the repository under your GitHub account.
2. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/aimed-agi-ultrasound.git
   ```

## Creating Branches
- Use descriptive names for your branches to indicate the feature or issue you are working on:
  ```bash
  git checkout -b feature/your-feature-name
  ```

## Committing Changes
- Commit your changes with clear and concise messages that explain your changes:
  ```bash
  git add .
  git commit -m "Add a brief description of your changes"
  ```

## Submitting Pull Requests
1. Push your changes to your forked repository:
   ```bash
   git push origin feature/your-feature-name
   ```
2. Navigate to the original repository and click on "Pull Requests."
3. Click the "New Pull Request" button, select your branch, and submit the pull request.
4. Include a description of the changes you made and any relevant information.

## Code of Conduct
We expect all contributors to follow our [Code of Conduct](CODE_OF_CONDUCT.md). Please read it to understand our expectations for participant behavior.

## Development Setup Instructions
To get started with the FillingVision project, follow these setup instructions:
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone the repository:
   ```bash
   git clone https://github.com/huang-carl/aimed-agi-ultrasound.git
   ```
3. Navigate to the project directory:
   ```bash
   cd aimed-agi-ultrasound
   ```
4. Install the dependencies:
   ```bash
   npm install
   ```
5. To run the development server:
   ```bash
   npm start
   ```

If you have any questions, feel free to reach out to the community or open an issue!
