# AIMED 充盈视界 - 答辩核心要点

**一句话介绍：** 用一杯"造影剂" + AI，把三甲医院的增强造影能力下沉到基层体检中心

---

## ❶ 我们做什么？（30 秒）

**产品：** 上消化道超声造影 AI 诊断系统

**核心公式：**
```
AIMED = 三剂 + 一法 + 一系统

三剂：胰腺型 + 胃肠型 + 胆管型 口服超声造影剂
一法：标准化充盈超声造影检查流程
一系统：AI 多智能体诊断平台
```

**简单说：** 患者喝一杯造影剂 → 做超声检查 → AI 辅助分析 → 10 分钟出报告

---

## ❷ 解决什么问题？（30 秒）

| 痛点 | 现状 | 我们的方案 |
|------|------|------------|
| **胃镜痛苦** | 侵入性、需麻醉、患者恐惧 | 无创、无辐射、口服即可 |
| **胰腺难查** | 胃肠气体干扰，显示率 60% | 充盈后显示率≥90% |
| **基层不会看** | 依赖医生经验，漏诊率高 | AI 辅助分析，准确率≥85% |
| **早筛成本高** | 胃镜 500-1000 元/次 | 超声造影 100-200 元/次 |

## ❾ 常见问题（Q&A）

**Q1：和胃镜比，优势在哪？**
A：无创 vs 侵入、100-200 元 vs 500-1000 元、普筛 vs 确诊，我们不是替代胃镜，而是做胃镜前的第一道筛查。

**Q2：AI 准确率多少？**
A：Phase 1 目标≥85%（内部测试），Phase 3 目标≥95%（临床验证）。当前 60 例数据，持续收集中。

**Q3：合规风险？**
A：Phase 1 科研工具（非临床诊断），Phase 2 二类器械备案，Phase 3 NMPA 三类证申报（2027 Q4）。

**Q4：市场竞争？**
A：差异化定位 - 普筛市场（体检中心 + 基层医院），避开胃镜红海，做增量市场。

**Q5：数据从哪来？**
A：共建医院提供 + 体检中心合作 + 科研数据采集，当前 60 例，目标 Phase 1 500 例。
