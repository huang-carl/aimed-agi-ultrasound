## 架构设计

```
┌─────────────────────────────────────────┐
│         AI Diagnosis Service            │
│      (多提供商自动切换层)                 │
└──────────────────┬──────────────────────┘
                   │
        ┌──────────┼──────────┐
        │          │          │
        ▼          ▼          ▼
   ┌────────┐ ┌────────┐ ┌────────┐
   │Dashscope│ │ Zhipu  │ │ NVIDIA │
   │ (优先级 1)│ │ (优先级 2)│ │ (优先级 3)│

        │          │          │
        └──────────┴──────────┘
                   │
        ┌──────────┴──────────┐

        └─────────────────────┘
```

## 测试结果

### 1. 胃诊断测试

**输入：**
```json
{
    "organ": "胃",
    "image_description": "胃体部黏膜增厚，表面不平整，可见多发结节样隆起，边界不清",
    "context": "患者，男，55岁，上腹不适3个月"
}
```

**输出：**
```json
{
    "success": true,
    "organ": "胃",
    "disease": "胃癌",
    "probability": 0.8,
    "suggestion": "建议进一步进行胃镜检查和病理活检以确认诊断",
    "mode": "ai",
    "model": "meta/llama-3.3-70b-instruct"
}
```

### 2. 胰腺诊断测试

**输入：**
```json
{
    "organ": "胰腺",
    "image_description": "胰腺头部可见低回声占位，大小约 2.5cm×2.0cm，边界不清，内部回声不均",
    "context": "患者，女，62岁，黄疸 2 周，体重下降 5kg"
}
```

**输出：**
```json
{
    "success": true,
    "organ": "胰腺",
    "disease": "胰腺癌",
    "probability": 0.8,
    "suggestion": "建议进行进一步的检查，如 CT 或 MRI，以确认诊断，并评估是否有淋巴结转移或远处转移",
    "mode": "ai",
    "model": "meta/llama-3.3-70b-instruct"
}
```
