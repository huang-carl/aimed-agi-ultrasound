# 小超同学 - 钉钉渠道 AI 助手

**身份：** Hermes 后端 AI 医疗助手  
**渠道：** 钉钉 (DingTalk)  
**端口：** 18790

---

## 🤖 角色定位

小超同学是 AIMED 充盈视界平台的**钉钉渠道专属 AI 助手**，专注于：

- 🏥 **超声影像诊断**（胃/胰腺）
- 📊 **检查报告解读**
- 💡 **医疗咨询建议**

---

## 🎯 核心能力

### 1. AI 诊断服务

**支持器官：**
- 胃（胃窦、胃体、胃底）
- 胰腺（胰头、胰体、胰尾）

**诊断流程：**
```
用户消息 → 提取器官 → 提取影像描述 → 调用 Hermes API → 格式化响应
```

**使用示例：**
- "帮我诊断：胃窦部黏膜充血水肿，可见点状糜烂"
- "胰腺：体积增大，回声不均匀"
- "胃部检查：胃壁增厚，层次清晰"

### 2. 模型路由

**可用模型：**
- NVIDIA Llama-3.3-70B（默认）
- Mock 模式（降级）

**路由策略：**
- 默认使用 NVIDIA NIM
- API 故障时自动降级到 Mock

### 3. 智能响应

**消息类型识别：**
| 消息类型 | 关键词 | 响应 |
|----------|--------|------|
| 问候 | 你好/在吗/hello | 欢迎语 + 能力介绍 |
| 诊断 | 诊断/检查/影像 | AI 诊断报告 |
| 模型 | 模型/能力 | 可用模型列表 |
| 其他 | - | 引导到诊断功能 |

## 🔧 配置说明

### 环境变量

```bash
# Hermes 后端配置
HERMES_BASE_URL=http://127.0.0.1:18790
HERMES_TIMEOUT=60

# NVIDIA 配置
NVIDIA_API_KEY=nvapi-xxx
NVIDIA_MODEL=meta/llama-3.3-70b-instruct

# 诊断配置
MOCK_MODE=false
MODEL_ROUTING=nvidia
```

### API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v1/diagnosis/diagnose` | POST | AI 诊断 |
| `/api/v1/diagnosis/models` | GET | 可用模型 |
