## 🎯 **数据标注规范**

### 器官标注

| 标签 | 说明 | 示例特征 |
|------|------|----------|
| **胃** | 胃部超声图像 | 胃壁层次、胃腔 |
| **胰腺** | 胰腺超声图像 | 胰头/体/尾、胰腺实质 |

### 充盈状态标注

| 标签 | 说明 | 判断标准 |
|------|------|----------|
| **已充盈** | 口服造影剂后 | 胃腔充盈良好、边界清晰 |
| **未充盈** | 未口服造影剂 | 气体干扰、边界不清 |

### 图像质量标注

| 标签 | 说明 |
|------|------|
| **good** | 图像清晰、无明显伪影 |
| **fair** | 图像可用、轻微伪影 |
| **poor** | 图像质量差、影响诊断 |
