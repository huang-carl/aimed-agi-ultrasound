## 🔧 **API 端点**

### 1. 上传病例

```bash
POST /api/v1/cases/upload
Content-Type: multipart/form-data

参数:
- organ: 器官类型（胃/胰腺）
- patient_id: 患者 ID（可选）
- patient_name: 患者姓名（可选）
- image: 病例图像（可选）
- image_description: 影像描述（可选）
```

**响应示例:**
```json
{
  "success": true,
  "case_id": "case_20260422234000_xxx",
  "message": "病例上传成功",
  "organ": "胃",
  "diagnosis": "慢性胃炎",
  "probability": 0.85
}
```

### 2. 获取病例列表

```bash
GET /api/v1/cases/list?organ=胃&limit=100
```

### 3. 获取单个病例

```bash
GET /api/v1/cases/{case_id}
```

### 4. 获取统计信息

```bash
GET /api/v1/cases/stats/statistics
```

**响应示例:**
```json
{
  "total_cases": 10,
  "organ_stats": {
    "胃": 6,
    "胰腺": 4
  },
  "category_stats": {
    "2026-04-22": 10
  },
  "recent_cases": [...],
  "storage": {
    "total_size_mb": 15.2,
    "file_count": 10,
    "by_organ": {
      "胃": {"count": 6, "size_mb": 9.1},
      "胰腺": {"count": 4, "size_mb": 6.1}
    }
  }
}
```

### 5. 导出报告

```bash
GET /api/v1/cases/export/report
```

### 6. 存储统计

```bash
GET /api/v1/cases/storage/stats
```

## 📊 **数据库表结构**

### cases 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 病例 ID（主键） |
| patient_id | TEXT | 患者 ID |
| patient_name | TEXT | 患者姓名 |
| organ | TEXT | 器官（胃/胰腺） |
| image_path | TEXT | 本地图像路径 |
| oss_url | TEXT | OSS 云存储 URL |
| image_description | TEXT | 影像描述 |
| diagnosis | TEXT | 诊断结果 |
| probability | REAL | 置信度 |
| image_quality | TEXT | 图像质量 |
| category | TEXT | 分类（日期） |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### categories 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 分类 ID |
| category_name | TEXT | 分类名称 |
| organ | TEXT | 器官类型 |
| date | TEXT | 日期 |
| case_ids | TEXT | 病例 ID 列表（JSON） |
| case_count | INTEGER | 病例数量 |
| created_at | TIMESTAMP | 创建时间 |

## ☁️ **阿里云 OSS 配置（可选）**

### 环境变量

```bash
ALIYUN_OSS_ENABLED=true
ALIYUN_OSS_BUCKET=aimed-cases
ALIYUN_OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
ALIYUN_ACCESS_KEY_ID=your_access_key
ALIYUN_ACCESS_KEY_SECRET=your_access_secret
```

### 存储策略

- **本地存储：** 默认启用，所有图像都保存
- **OSS 云存储：** 可选，启用后自动同步到云端
- **混合模式：** 本地 + 云端双备份
