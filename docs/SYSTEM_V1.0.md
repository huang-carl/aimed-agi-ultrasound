# AIMED 充盈视界 - 系统 V1.0 定义

**版本：** V1.0  
**创建时间：** 2026-04-27

## 一、系统架构

```
AIMED 充盈视界系统 V1.0
├── 1. 口服超声造影剂（三种型号）
│   ├── 胰腺型
│   ├── 胃肠型
│   └── 胆管型
├── 2. 标准化检查流程
│   ├── 影像采集规范
│   └── 操作流程 SOP
├── 3. AI 多智能体诊断系统（Hermes）
│   ├── 前端展示层（OpenClaw）
│   ├── 后端服务层（Hermes API）
│   └── AI 模型层（阿里云百炼 + NVIDIA NIM）
└── 4. 数据管理平台
    ├── 病例库（500+ 标注病例目标）
    ├── 知识库（23+ 文档）
    └── 训练数据管理
```

## 三、技术栈

### 3.1 前端
- **框架：** 原生 HTML/CSS/JS
- **部署：** Nginx + HTTPS
- **域名：** aius.xin

### 3.2 后端
- **框架：** FastAPI + Uvicorn
- **端口：** 18792
- **AI 模型：** 阿里云百炼（qwen3.5-plus/coder-plus/max + kimi-k2.5）

### 3.3 基础设施
- **服务器：** 阿里云 ECS（8.141.91.165）
- **SSL：** Let's Encrypt
- **服务管理：** systemd + 自动重启
