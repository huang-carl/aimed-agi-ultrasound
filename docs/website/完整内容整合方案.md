## 🔧 后端 API（Hermes 18790 端口）

### API v1 路由

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/v1/upload` | POST | 病例上传（支持器官/充盈状态/诊断信息） |
| `/api/v1/cases/list` | GET | 病例列表查询 |
| `/api/v1/cases/{id}` | GET | 单个病例详情 |
| `/api/v1/cases/stats/statistics` | GET | 病例统计信息 |
| `/api/v1/diagnosis/stomach` | POST | 胃诊断（双模型路由） |
| `/api/v1/diagnosis/pancreas` | POST | 胰腺诊断（双模型路由） |
| `/api/v1/conductor/analyze` | POST | 总指挥多智能体分析 |
| `/api/v1/report/generate` | POST | 诊断报告生成 |

### Multi-Agent 系统

```
┌─────────────────────────────────────────────────────────┐
│                    总指挥智能体                          │
│              (任务分发/结果汇总/质量控制)                │
└────────────────────┬────────────────────────────────────┘
                     │
         ┌───────────┼───────────┐
         │           │           │
         ▼           ▼           ▼
┌─────────────┐ ┌─────────┐ ┌──────────┐
│ 胃诊断智能体 │ │胰腺诊断 │ │ 报告生成 │
│             │ │ 智能体  │ │ 智能体   │
└─────────────┘ └─────────┘ └──────────┘
```

### 双模型路由策略

| 模型 | 场景 | 特点 |
|------|------|------|
| **NVIDIA NIM (Llama-3.3-70B)** | 长文本 >50K | 高性能，广泛知识 |
| **阿里云百炼 (Qwen-Plus)** | 短文本/快速响应 | 中文优化，成本低 |
