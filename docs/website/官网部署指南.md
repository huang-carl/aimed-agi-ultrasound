### Nginx 配置示例

```nginx
server {
    listen 80;
    server_name aius.xin www.aius.xin;
    
    # 重定向到 HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name aius.xin www.aius.xin;
    
    # SSL 证书配置
    ssl_certificate /etc/nginx/ssl/aius.xin.crt;
    ssl_certificate_key /etc/nginx/ssl/aius.xin.key;
    
    # 官网静态文件
    location / {
        root /root/.openclaw/workspace/static;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理（Hermes 后端）
    location /api/ {
        proxy_pass http://127.0.0.1:18790;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
    
    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

## 🚀 部署步骤

### 1. 上传文件到服务器
```bash
# 确保工作空间在服务器上
cd /root/.openclaw/workspace/static
ls -la index.html
```

### 2. 配置 Nginx
```bash
# 编辑 Nginx 配置
sudo nano /etc/nginx/sites-available/aius.xin

# 启用配置
sudo ln -s /etc/nginx/sites-available/aius.xin /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 3. 配置 SSL 证书
```bash
# 使用 Let's Encrypt 免费证书
sudo certbot --nginx -d aius.xin -d www.aius.xin
```

### 4. 验证部署
```bash
# 测试 HTTP
curl -I http://aius.xin

# 测试 HTTPS
curl -I https://aius.xin
```
