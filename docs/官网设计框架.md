### 3. 公司介绍
**背景：** 浅灰色

**布局：** 左右两栏（文字 + 图片）

**内容：**
```
H2: 公司介绍
副标题：专注智慧医疗，赋能基层健康

H3: 阿尔麦德智慧医疗（湖州）有限公司

段落 1:
  阿尔麦德智慧医疗（湖州）有限公司是一家专注于医学影像 AI 诊断的创新型企业，
  致力于将先进的超声造影技术与人工智能深度融合，
  打造面向基层医疗机构的胃胰疾病早筛解决方案。

段落 2 (核心公式):
  核心公式：AIMED = 三剂 + 一法 + 一系统
  
  三剂：胰腺型 + 胃肠型 + 胆管型口服超声造影剂
  一法：标准化充盈超声造影检查流程
  一系统：AI 多智能体诊断平台（OpenClaw 前端 + Hermes 后端）

段落 3 (简单流程):
  简单说：患者喝一杯造影剂 → 做超声检查 → AI 辅助分析 → 10 分钟出报告

段落 4 (双智能体架构):
  采用双智能体架构：前端 OpenClaw（钉钉/飞书双渠道）+ 后端 Hermes（AI 多智能体诊断），
  共享同一工作空间、配置、数据和 GitHub 仓库，内容/数据完全同步。

共建单位框:
  🤝 共建单位：阿尔麦德智慧医疗（湖州）有限公司、南京大学、湖州师范学院附属第一医院

统计数据:
  101+ 标注病例（目标 500 例）
  胃 + 胰腺 双核心 P0 优先
  85%+ Phase 1 准确率目标
```

## 📊 四、响应式设计

### 断点
- **桌面端：** > 1024px（完整布局）
- **平板端：** 768px - 1024px（2 列变 1 列）
- **手机端：** < 768px（导航隐藏，单列布局）

### 适配规则
| 元素 | 桌面端 | 平板端 | 手机端 |
|------|--------|--------|--------|
| 导航栏 | 完整显示 | 完整显示 | 隐藏（汉堡菜单） |
| 公司介绍 | 2 列 | 2 列 | 1 列 |
| 联合实验室 | 2 列 | 2 列 | 1 列 |
| 技术架构 | 3 列 | 2 列 | 1 列 |
| 核心优势 | 4 列 | 2 列 | 1 列 |
| 页脚 | 3 列 | 2 列 | 1 列 |

## 🔧 六、技术实现

### 前端技术栈
- **HTML5** - 语义化标签
- **CSS3** - 原生 CSS（无框架）
- **JavaScript** - 原生 JS（仅平滑滚动）

### 性能优化
- 内联 CSS（单文件部署）
- 图片懒加载
- 最小化 JS

### SEO 优化
- 语义化 HTML 标签
- Meta 描述与关键词
- 结构化数据（Schema.org）
