## 🔒 SSL 证书说明

### 当前状态
- **类型：** 自签名证书
- **有效期：** 365 天
- **适用范围：** aius.xin, www.aius.xin
- **警告：** 浏览器会显示"不安全"提示

### 替换为正式证书（推荐）

**方案 A：阿里云免费 SSL 证书**
1. 登录阿里云控制台 → SSL 证书服务
2. 申请免费证书，域名填写 `aius.xin`
3. DNS 验证（添加 TXT 记录）
4. 下载 Nginx 格式证书
5. 上传到服务器：
   ```bash
   /etc/nginx/ssl/aius.xin.crt
   /etc/nginx/ssl/aius.xin.key
   ```
6. 重载 Nginx：`/usr/sbin/nginx -s reload`

**方案 B：Let's Encrypt（需开放 80 端口）**
```bash
certbot --nginx -d aius.xin -d www.aius.xin
```
