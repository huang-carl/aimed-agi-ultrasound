## 🎯 核心功能

### 1. AI 诊断服务

**支持器官：**
- 胃（胃窦、胃体、胃底）
- 胰腺（胰头、胰体、胰尾）

**诊断输出：**
1. 诊断结论
2. 置信度（0-1）
3. 鉴别诊断
4. 进一步检查建议
5. 治疗建议

### 2. 双模型路由

```python
# 智能路由策略
if context_length > 50000:
    return NVIDIA  # 长文本用 NVIDIA (1M 上下文)
elif aliyun_available:
    return 阿里云   # 常规用阿里云
elif nvidia_available:
    return NVIDIA  # 降级到 NVIDIA
else:
    return Mock    # 最终降级
```

### 3. 自动降级机制

| 场景 | 降级策略 |
|------|----------|
| NVIDIA API 故障 | → Mock 模式 |
| 阿里云 API 故障 | → NVIDIA/Mock |
| 无效输入 | → HTTP 400 + 提示 |
