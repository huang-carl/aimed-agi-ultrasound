## 一、页面结构

```
AIMED 网站 V1.0
├── 1. 官网首页 (index.html)
│   ├── 品牌展示区
│   ├── 快速体验区（iframe 嵌入 Demo）
│   └── 平台入口
├── 2. 诊断演示页 (demo.html)
│   ├── AI 模型选择
│   ├── 患者信息录入
│   ├── 影像上传
│   ├── AI 分析进度
│   ├── 医生标注
│   └── 报告生成
├── 3. AI 平台 (portal/)
│   ├── 登录页
│   ├── 医生工作台
│   ├── 患者咨询台
│   └── 开发者中心
└── 4. 其他页面
    ├── 产品页 (products.html)
    ├── 解决方案页 (solutions.html)
    └── 技术页 (technology.html)
```

## 二、核心功能

### 2.1 诊断演示功能
| 功能 | 描述 | 状态 |
|------|------|------|
| 模型选择 | 3 种 AI 模型可选 | ✅ |
| 患者信息 | 姓名/年龄/性别/器官 | ✅ |
| 高危评估 | 自动识别高危人群 | ✅ |
| 影像上传 | 最多 10 张图像 + 3 条视频 | ✅ |
| AI 分析 | 实时进度 + 日志显示 | ✅ |
| 医生标注 | Canvas 绘制/修改标注 | ✅ |
| 报告生成 | Su-RADS 分类报告 | ✅ |

|------|------|------|
| 首页 | / | ✅ |
