❓ 常见问题

AIMED 平台的诊断结果可以作为临床诊断依据吗？

▼

目前 Phase 1 阶段，AI 诊断结果仅供科研参考和技术验证，不能直接作为临床诊断依据。Phase 2 完成二类器械备案后，可在试点医院作为辅助诊断工具使用。Phase 3 取得 NMPA 三类证后，方可作为临床诊断辅助依据。详见 合规边界说明。

如何调用诊断 API？需要什么认证？

▼

调用诊断 API 需要：1）在平台注册账号并登录；2）在开发者中心获取 API Key；3）在请求 Header 中携带 Authorization: Bearer {api_key}。详细文档请参考 API 文档页面。免费额度内可直接使用。

口服超声造影剂（充盈视界）如何使用？

▼

充盈视界是口服超声造影剂，使用前需空腹 6-8 小时。检查前 15 分钟口服 200-300ml 造影剂溶液，根据检查部位选择不同体位。胃部检查建议仰卧位 + 左侧卧位，胰腺检查建议仰卧位 + 半卧位。详见 临床使用指南。

AI 诊断的准确率如何？支持哪些疾病？

▼

目前支持胃部（胃癌、胃溃疡、胃炎）和胰腺（胰腺癌、胰腺炎）疾病的 AI 辅助分析。内部测试准确率 ≥85%（Phase 1 目标）。随着训练数据积累（目标 500+ 标注病例），准确率将持续提升。详见 算法性能报告。

上传的病例数据如何保证安全？

▼

平台采用多重安全措施：1）基于 FISCO BCOS 的区块链 DID 身份验证；2）数据传输全程 HTTPS 加密；3）病例数据脱敏处理；4）数字资产管理智能合约控制访问权限；5）符合 HIPAA/GDPR 数据保护标准。详见 安全白皮书。

API 调用频率有限制吗？

▼

免费用户：100 次/天；认证用户：1000 次/天；机构用户：10000 次/天。超出限制将返回 429 错误。如需更高配额，请联系 admin@aius.xin 申请。并发请求限制：最多 5 个同时请求。
