阿尔麦德智慧医疗（湖州）有限公司

阿尔麦德智慧医疗（湖州）有限公司是一家专注于医学影像 AI 诊断的创新型企业， 致力于将先进的超声造影技术与人工智能深度融合，打造面向基层医疗机构的胃胰疾病早筛解决方案。

采用双智能体架构：前端 OpenClaw（钉钉/飞书双渠道）+ 后端 Hermes（AI 多智能体诊断）， 共享同一工作空间、配置、数据和 GitHub 仓库，内容/数据完全同步。 通过自主研发的口服超声造影剂（胰腺型/胃肠型/胆管型），让复杂的增强造影检查变得简单、便捷、可及。

核心技术

三剂 + 一法 + 两系统，构建精准诊断护城河

💊

口服超声造影剂（三剂）

自主研发的三种型号口服超声造影剂（胰腺型/胃肠型/胆管型），安全无创， 显著提升胃/胰腺/胆管超声成像质量，使粘膜下病变清晰可见， 为 AI 诊断提供高质量图像基础。

🤖

AI 多智能体诊断系统（Hermes）

后端 Hermes 系统，基于 NVIDIA NIM + 阿里云百炼双引擎，胃/胰腺双专科 AI 智能体协同工作， 聚焦胃P0+胰腺P0双核心， 按序延伸至胆管P1/食道P2/贲门P3/十二指肠P4， 确保诊断的专业性和准确性。

📱

智能交互平台（OpenClaw）

前端 OpenClaw 系统，支持钉钉/飞书双渠道， 提供用户交互、日常咨询、工作空间管理等功能， 与 Hermes 后端数据完全同步，前后端分离架构。
