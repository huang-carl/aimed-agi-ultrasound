阿尔麦德智慧医疗（湖州）有限公司

阿尔麦德智慧医疗（湖州）有限公司是一家专注于医学影像 AI 诊断的创新型企业， 致力于将先进的超声造影技术与人工智能深度融合，打造面向基层医疗机构的胃胰疾病早筛解决方案。

公司采用双智能体架构：前端 OpenClaw（钉钉/飞书双渠道）+ 后端 Hermes（AI 多智能体诊断）， 共享同一工作空间、配置、数据和 GitHub 仓库，内容/数据完全同步。

核心公式：AIMED = 二剂 + 一法 + 一系统

二剂：胰腺型 + 胃肠型口服超声造影剂
一法：标准化充盈超声造影检查流程
一系统：AI 多智能体诊断平台（OpenClaw 前端 + Hermes 后端）

简单说：患者喝一杯造影剂 → 做超声检查 → AI 辅助分析 → 10 分钟出报告
