# 数据目录说明

## 目录结构

```
data/
├── samples/          # 测试样本目录
│   ├── README.md     # 样本说明
│   └── .gitkeep
├── reports/          # 生成的 PDF 报告
│   └── .gitkeep
└── README.md         # 本文件
```

## 各目录用途

### samples/ - 测试样本
存放胃和胰腺超声影像测试样本，用于开发和测试。

### reports/ - 诊断报告
存放 AI 生成的 PDF 诊断报告文件。

## 安全要求

1. **生产环境** - 患者数据必须加密存储
2. **访问控制** - 仅限授权人员访问
3. **定期备份** - 建议每日备份
4. **审计日志** - 记录所有数据访问操作
